
# 使用nodejs实现web数据统计，移动应用统计分析


|关键字 | 类型  | 语义  |
|-  | - | - |
|pv | 页面被打开的次数 |	每个页面被打开一次，计为1次访问	|
|uv | 访客1天内去除重复访问次数 |	1天内相同访客访问次数只计1次访问	|
|vv | 访客1天内访问次数 | 从访客来到您网站到最终关闭网站的所有页面离开，计为1次访问	|
|iv | 1天内访问的IP数 | - |
